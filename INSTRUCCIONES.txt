🍪 GALLETAS KATI - INSTRUCCIONES DE EJECUCIÓN
================================================================

PASO 1: Abrir terminal en el directorio del proyecto
   cd "c:\Users\cuent\Galletas Kati"

PASO 2: Ejecutar script de configuración
   .\CONFIGURAR_BD.bat

O ALTERNATIVAMENTE, ejecutar comandos uno por uno:

PASO 2a: Crear migraciones
   python manage.py makemigrations notifications --verbosity=2

PASO 2b: Aplicar migraciones
   python manage.py migrate --verbosity=2

PASO 2c: Verificar migraciones
   python manage.py showmigrations

PASO 2d: Iniciar servidor
   python manage.py runserver 8002
